Loess (Locally Weighted Scatterplot Smoothing)
==========================================================================

This is an implementation of the Loess algorithm in pure numpy.
The algorithm is introduced and described in `Cleveland (1979) <https://sites.stat.washington.edu/courses/stat527/s14/readings/Cleveland_JASA_1979.pdf>`_
